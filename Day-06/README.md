# Day 06 – Linux Fundamentals: Read and Write Text Files

## Task

Today I practiced file handling in Linux by creating, reading, and modifying text files.

---

## What I Learned

- Creating files
- Writing data into files
- Appending content
- Reading file content

---

## Commands Practiced

- `touch` → Create file  
- `echo` → Write content  
- `cat` → Read file  
- `head` → View beginning  
- `tail` → View end  

---

## Hands-on Practice

- Created text files using `touch`
- Added content using `echo`
- Viewed file content using `cat`
- Used `head` and `tail` for partial reading

---

## Key Takeaway

File handling is a fundamental skill in Linux and is widely used in scripting and automation.
