## Commands Practiced

### Navigation Commands

- `pwd` → Show current directory  
- `ls` → List files and folders  
- `cd` → Change directory  

---

### File & Directory Commands

- `mkdir` → Create a directory  
- `rmdir` → Remove empty directory  
- `touch` → Create a file  
- `rm` → Delete a file  

---

### File Viewing

- `cat` → View file content  

---

## Hands-on Practice
